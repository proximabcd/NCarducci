## Débogage et optimisation d'un site de photographe ##

Ce projet avait pour objectif d'améliorer le référencement du site d'une photographe. Certaines anomalies du code devaient également être déboguées.

Principales opérations effectuées :

-> audit avec Lighthouse

-> optimisation de la taille des images avec Gulp, Sharp et les attributs "srcset" et "images"

-> chargement inline du CSS critique avec Critical

-> ajout de métabalises Open Graph associées à des textes accrocheurs

-> ajout de métabalises "title" et "description" associées à un texte alliant plusieurs mots-clés

-> amélioration de l'accessibilité du site en renommant certaines balises

-> débogage avec les DevTools
